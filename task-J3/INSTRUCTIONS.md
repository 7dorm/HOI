# Краткая инструкция по использованию

## Быстрый старт

### 1. Запуск сервера

```bash
java -jar Task_J3-server.jar <student_id> [port]
```

Пример:
```bash
java -jar Task_J3-server.jar IvanIvanov 8080
```

### 2. Сборка клиента

```bash
cd task_3
mvn clean package
```

После сборки будет создан файл `target/Task_J3-client.jar`

### 3. Запуск клиента

```bash
java -jar target/Task_J3-client.jar <student_id> [host] [port]
```

Пример:
```bash
java -jar target/Task_J3-client.jar IvanIvanov localhost 8080
```

Или используйте скрипт:
```bash
./run.sh IvanIvanov localhost 8080
```

## Параметры

- `student_id` (обязательный) - имя и фамилия латиницей
- `host` (опциональный) - хост сервера, по умолчанию `localhost`
- `port` (опциональный) - порт сервера, по умолчанию `8080`

## Что делает программа

1. Начинает обход с корневого пути "/"
2. Выполняет HTTP GET запрос к каждому пути
3. Парсит JSON ответ и извлекает сообщение и список successors
4. Параллельно обходит все successors используя виртуальные потоки Java 21
5. Собирает все сообщения
6. Сортирует их в лексикографическом порядке
7. Выводит результат

## Требования

- Java 21+ (для виртуальных потоков)
- Maven 3.6+ (для сборки)
- Запущенный сервер Task_J3-server.jar
