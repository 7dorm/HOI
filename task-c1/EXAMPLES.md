# Примеры использования функции generate-strings

## Пример 1: Базовый пример из задания

```clojure
(generate-strings ["a" "b" "c"] 2)
```

**Ожидаемый результат:**
```
("ab" "ac" "ba" "bc" "ca" "cb")
```

**Объяснение:**
- Алфавит содержит 3 символа: "a", "b", "c"
- Нужны строки длины 2 без повторяющихся подряд символов
- Для каждого символа добавляем все остальные символы:
  - "a" → "ab", "ac"
  - "b" → "ba", "bc"
  - "c" → "ca", "cb"
- Итого: 6 строк

## Пример 2: Строки длины 3

```clojure
(generate-strings ["a" "b" "c"] 3)
```

**Ожидаемый результат:**
```
("aba" "abc" "aca" "acb" "bab" "bac" "bca" "bcb" "cab" "cac" "cba" "cbc")
```

**Объяснение:**
- Начинаем со строк длины 1: "a", "b", "c"
- Шаг 1 (длина 2):
  - "a" → "ab", "ac"
  - "b" → "ba", "bc"
  - "c" → "ca", "cb"
- Шаг 2 (длина 3):
  - "ab" → "aba", "abc" (последний символ "b", добавляем "a" и "c")
  - "ac" → "aca", "acb" (последний символ "c", добавляем "a" и "b")
  - "ba" → "bab", "bac" (последний символ "a", добавляем "b" и "c")
  - "bc" → "bca", "bcb" (последний символ "c", добавляем "a" и "b")
  - "ca" → "cab", "cac" (последний символ "a", добавляем "b" и "c")
  - "cb" → "cba", "cbc" (последний символ "b", добавляем "a" и "c")
- Итого: 12 строк (3 * 2 * 2 = 12)

## Пример 3: Алфавит из 2 символов

```clojure
(generate-strings ["x" "y"] 4)
```

**Ожидаемый результат:**
```
("xyxy" "yxyx")
```

**Объяснение:**
- Алфавит содержит только 2 символа: "x" и "y"
- Для строки длины 4 возможны только чередующиеся паттерны:
  - "xyxy" (начинается с "x")
  - "yxyx" (начинается с "y")
- Итого: 2 строки (2 * 1 * 1 * 1 = 2)

## Пример 4: Алфавит из 4 символов

```clojure
(generate-strings ["a" "b" "c" "d"] 2)
```

**Ожидаемый результат:**
```
("ab" "ac" "ad" "ba" "bc" "bd" "ca" "cb" "cd" "da" "db" "dc")
```

**Объяснение:**
- Алфавит содержит 4 символа
- Для каждого символа добавляем все остальные 3 символа
- Итого: 4 * 3 = 12 строк

## Математическая формула

Для алфавита размера `k` и длины строки `n`:

**Количество строк = k × (k-1)^(n-1)**

Примеры:
- k=3, n=2: 3 × 2^1 = 6 строк
- k=3, n=3: 3 × 2^2 = 12 строк
- k=2, n=4: 2 × 1^3 = 2 строки
- k=4, n=2: 4 × 3^1 = 12 строк

## Граничные случаи

### N=1
```clojure
(generate-strings ["a" "b" "c"] 1)
=> ("a" "b" "c")
```
Просто все символы алфавита.

### N=0 или отрицательное
```clojure
(generate-strings ["a" "b" "c"] 0)
=> ()
```
Пустой список.

### Пустой алфавит
```clojure
(generate-strings [] 2)
=> ()
```
Пустой список.
